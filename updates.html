Version 1.6.5 September 12th, 2011
  • Feature: Add ability to freeze GIF images and allow the user to play it again when hovering it.
  • Feature: Allow individual circles (streams) to be filtered.
Version 1.6.0 September 12th, 2011
  • Feature: Ability to filter out animated gifs. Visit the options screen and under the control section, select the checkbox that filters gifs.
  • Bugfix: Fix spelling error in the options screen. Thanks to Lina De Martínez.
Version 1.5.0 September 11th, 2011
  • Feature: Support Regular Expressions for Exclusions and Inclusions. By wrapping the keyword with forward slashes. For example, matching only hashtags would be /\\s#(.*)\\s/. If the slashes are not there, it will do a normal keyword filter.
  • Feature: Add ability to turn on and off the filtering from options. This requires a manual stream refresh everytime you toggle this.
  • Feature: Implement Mark as Read for the posts filtered. A new link will appear in the popup that when clicked, it will reset the counter and wipe the posts.
  • Feature: Implement auto reload setting, where you can automatically load the new settings when you complete the save in options.
  • Improvement: Improve the popup stats UI to discover the inclusions and exclusions faster with some text color.
  • Improvement: Improve the options page UI to better visualize the the save button on the bottom.
  • Improvement: Improve the dark icon in the browser by Andy Shaw.
  • Bugfix: Use exclusion filters before inclusion.
  • Bugfix: Fix the bug where the popup contents and counter doesn't reset when no results appear.
  • Bugfix: Fix GitHub URL pointing to old repository.
